# Duct Manufacturing Production Dashboard

Simple Streamlit mock dashboard for tracking duct manufacturing jobs, machine utilization, WIP, and completed work.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files

- `app.py` - Streamlit dashboard
- `machines.csv` - mock machine status and utilization
- `jobs.csv` - mock production jobs
- `requirements.txt` - dependencies

## Future version

Replace CSV files with a small database and let operators update job status from a phone, tablet, or shop-floor computer.
